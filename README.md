# Cashfolio

## Development

Install dependencies:

```sh
npm install
```

Run the database:

```sh
npm run db
```

Start the app in the dev server:

```sh
npm run dev
```

### Playwright Tests

Start Playwright in UI Mode:

```sh
npm run e2e
```

### Storybook

Start Storybook by

```sh
npm run storybook
```

## Deployment

Trigger Fly deployment from the local machine:

```sh
npm run deploy
```
