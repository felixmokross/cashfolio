# Cashfolio

## Development

Install dependencies:

```sh
npm run dev
```

Run the database:

```sh
dockercompose up
```

Start the app in the dev server:

```sh
npm run dev
```

## Deployment

Trigger Fly deployment from the local machine:

```sh
npm run deploy
```
